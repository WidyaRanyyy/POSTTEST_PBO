## Petunjuk Membaca Dokumentasi

Dokumentasi yang berkaitan dengan proyek ini dapat ditemukan pada folder **assets**.

Setiap file dokumentasi di dalam folder tersebut telah diberi **nomor penamaan** untuk memudahkan urutan pembacaan.  
Silakan membaca dokumentasi **sesuai dengan nomor yang tertera pada nama file**, dimulai dari nomor terkecil hingga terbesar agar alur informasi dapat dipahami dengan baik.

Contoh urutan pembacaan:
1. nama dokumentasi (01)
2. nama_dokumentasi (02)
3. nama_dokumentasi (03)
4. dan seterusnya.

Dengan mengikuti urutan nomor tersebut, pembaca dapat memahami penjelasan proyek secara **sistematis dan berurutan**.
